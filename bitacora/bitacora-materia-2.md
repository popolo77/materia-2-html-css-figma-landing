# Bitácora - Materia 2: HTML, CSS, Responsive Design y Figma Handoff

Objetivo de la materia
Aprender a maquetar interfaces web profesionales, responsivas y accesibles basándonos en diseños de Figma, utilizando HTML5 semántico y CSS moderno, consolidando el flujo de desarrollo y el despliegue progresivo.

Esta bitácora registra el avance clase por clase de la Materia 2 del Programa Complementario de Formación Frontend.

Clase 1 - Entorno Cloud y Estructura Semántica Básica
En esta clase se trabajó la configuración del entorno de desarrollo remoto y el uso de etiquetas semánticas para estructurar la base de la landing page de Uber Tracker.
Aprendizajes principales
* Configuración y uso de GitHub Codespaces como entorno de desarrollo en la nube.
* Uso de etiquetas semánticas estructurales (header, main, section, footer) para definir la arquitectura de la información.
* Jerarquía de contenidos mediante títulos (h1 a h6), párrafos, listas y enlaces.
Evidencia generada
* Archivo clases/clase-01-html-semantico.md creado.
* Estructura inicial del proyecto dentro de practica/practica-01-estructura-html/index.html.

Clase 2 - Formularios, Tablas y Metadatos de SEO
En esta clase se implementaron elementos avanzados de captura de datos, organización de información compleja y metadatos para optimización en motores de búsqueda.
Aprendizajes principales
* Configuración del bloque head con metadatos técnicos (description, keywords, author) para SEO básico.
* Estructura de elementos de formularios (form, input, label, textarea, select, button).
* Criterio de accesibilidad mediante la vinculación unívoca entre label e input usando los atributos for e id.
* Organización de datos estructurados mediante tablas (table, thead, tbody, tr, th, td).
Evidencia generada
* Archivo clases/clase-02-formularios-seo.md creado.
* Sección de contacto, formulario de alta y tabla de simulación de rendimiento semanal integrados en index.html.

Clase 3 - CSS Base y Variables Globales
En esta clase se desvinculó el diseño del archivo estructural enlazando una hoja de estilos externa, se definieron variables de identidad visual y se aplicaron reglas de Box Model a contenedores, tablas y formularios.
Aprendizajes principales
* Creación y enlace de archivos CSS externos mediante la etiqueta link en el head.
* Uso del selector global :root para declarar Custom Properties (variables CSS) reutilizables y escalables.
* Manipulación de la anatomía del Box Model: uso estratégico de padding para aire interno, margin para separación externa y box-sizing: border-box para prevenir desbordamientos involuntarios.
* Aplicación de pseudo-clases de interacción como :hover en filas de tablas y :focus en elementos de formulario para mejorar la accesibilidad visual.
Evidencia generada
* Archivo practica/practica-01-estructura-html/styles.css creado, estructurado y conectado.
* Interfaz de la landing page adaptada con fondo corporativo, tarjetas de sección alineadas, tabla financiera estilizada y formulario de contacto responsivo.
